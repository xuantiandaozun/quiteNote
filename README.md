# QuickNote - 随手记

> 极简的网页随手记工具，打开即记

## ✨ 特性

- 🚀 **即开即记** - 页面加载时自动聚焦，无需额外操作
- 💾 **本地存储** - 使用 IndexedDB 本地存储，数据永不丢失
- 🔄 **自动保存** - 智能防抖保存，停止输入后自动保存
- 🌓 **深色/浅色模式** - 支持主题切换，跟随系统偏好
- 📱 **PWA 支持** - 可安装到桌面/主屏幕，支持离线使用
- 🔗 **多标签同步** - 使用 BroadcastChannel API 实现多标签页实时同步
- 📋 **一键复制** - 快速复制全文到剪贴板
- 🗑️ **安全清除** - 二次确认机制，防止误操作
- 📊 **字数统计** - 实时显示字符数量
- 🎨 **现代设计** - 精致的排版和微交互动画
- 📝 **多笔记管理** - 支持创建、切换、搜索和删除多个笔记

## 🛠️ 技术栈

- **框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **存储**: IndexedDB (idb-keyval)
- **PWA**: vite-plugin-pwa
- **样式**: Modern CSS (CSS Variables, Flexbox, Grid)

## 📝 使用说明

1. **开始记录**: 打开应用后，输入框会自动获取焦点，直接开始输入即可
2. **自动保存**: 停止输入约 800ms 后会自动保存到本地
3. **管理笔记**: 点击左上角菜单按钮打开笔记列表侧边栏
4. **新建笔记**: 点击底部工具栏的"新建"按钮或侧边栏中的"新建笔记"
5. **切换笔记**: 在侧边栏点击任意笔记进行切换
6. **搜索笔记**: 在侧边栏搜索框输入关键词搜索笔记
7. **复制内容**: 点击底部"复制"按钮可将全文复制到剪贴板
8. **清除内容**: 点击"清除"按钮，再次点击确认后清除当前笔记内容
9. **切换主题**: 点击右上角的主题图标切换深色/浅色模式
10. **离线使用**: 首次访问后，即使断网也可以正常使用
11. **安装到桌面**: 在浏览器中选择"安装应用"或"添加到主屏幕"
